Sudokito wapito

Revienta al sudoku

Practica 3 PCW

La práctica consiste en implementar un Sudoku con el que el usuario pueda interactuar con el ratón para jugar. Para ello hemos utilizado HTML ,CSS y JavaScript, de este último la herramienta canvas ha sido la principal para el funcionamiento del juego. Además hecho que el sitio web sea responsive.

Problemas e incompatibilidades

Según el navegador que utilizaramos en algunos casos, las dimensiones o la posicion de ciertos elementos variaba. Despúes de estar probando, hemos corregido algunos de esos errores mediante CSS. Sobre todo a la hora del atributo height de los elementos, que nos dio problemas. También nos dio algún que otro problema el appendChild de JavaScript que no lo implementabamos bien para algunos objetos y nos fallaba el canvas por ello.
